Comandos del bot

!tts <texto> → manda texto a TTS.

!status → muestra estado de la cola/bot.

!skip → salta el mensaje actual.

!replay last | !replay <id> → reencola un mensaje.

!cancel <id> → marca SKIPPED el mensaje.

!delete <id> → borra el mensaje del historial.

!restore <id> → vuelve a encolar un SKIPPED.

!help → habla solo los comandos disponibles para viewers.

!voice <alias|voz> / !ttsVoice <alias|voz> / !ttsVoiceAlvaro → cambia la voz del TTS.

!preset <nombre> → cambia el preset emocional.
